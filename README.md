
# House Price Prediction using Feature Scaling and Model Comparison

## Artificial Intelligence & Machine Learning Internship – Task 2

### Project Description

This project focuses on enhancing the performance of a House Price Prediction system through feature scaling, model optimization, and comparative analysis of multiple machine learning algorithms. The objective was to identify the most effective model for predicting house prices using the California Housing Dataset.

---

## Objectives

- Apply feature scaling techniques to improve model performance.
- Train and evaluate multiple machine learning algorithms.
- Compare model performance using standard evaluation metrics.
- Select the most suitable model based on predictive accuracy.
- Understand real-world machine learning workflow and model optimization.

---

## Dataset Information

**Dataset:** California Housing Dataset

**Target Variable:** House Price

**Input Features:**
- Median Income
- House Age
- Average Rooms
- Average Bedrooms
- Population
- Average Occupancy
- Latitude
- Longitude

**Total Records:** 20,640

---

## Machine Learning Workflow

1. Dataset Loading

---

## Models Implemented

- Linear Regression
- Ridge Regression
- Decision Tree Regressor

---

## Performance Comparison

| Model | RMSE | R² Score |
|---------|---------|---------|
| Linear Regression | 0.745581 | 0.575788 |
| Ridge Regression | 0.745554 | 0.575819 |
| Decision Tree Regressor | 0.724234 | 0.599732 |

---

## Best Performing Model

### Decision Tree Regressor

**Performance Metrics**

- RMSE: 0.724234
- R² Score: 0.599732

The Decision Tree Regressor achieved the lowest prediction error and the highest explanatory power among all evaluated models, making it the best-performing model for this task.
